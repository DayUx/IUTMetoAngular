Faire une recherche :
Se rendre sur l'accueil, dans l'input recherche saisir le nom de la ville et appuyer ensuite sur la loupe, une liste de "cartes" apparait pour consulter la meteo d'une ville cliquez sur la carte correspondante.
Enregistrer un favoris :
dans l'onglet accueil, apres avoir fait une recherche cliquez sur le bouton coeur en bas a droite d'une des cartes.
Consulter les favoris : 
sur l'onglet accueil, cliquez sur le bouton coeur en bas a droite de l'ecran, un onglet s'affiche avec la liste des villes (celles-ci peuvent etre supprimmees en appuyant sur la croix).


Le site utilise AngularJs, font-awesome, API Openweathermap et Weather Icon.
